## 摘要
从单张非约束照片重建逼真的3D人脸化身具有挑战性：前馈3D高斯泼溅（3DGS）模型在分布外输入上性能下降，而预训练扩散模型生成高保真图像但缺乏多视角一致性。我们观察到这些范式本质上是互补的：显式3D表示保证几何一致性，而2D扩散先验确保照片真实感。基于此，我们提出SplatShot，一种无训练框架，直接在去噪过程中耦合这些表示。给定一个基础3DGS人脸模型和单张参考图像，我们使用每步3D反馈循环联合去噪所有目标视角。在每个时间步，我们从噪声潜变量预测干净图像，将这些多视角预测重新拟合到3DGS，并将3DGS重渲染与2D预测之间的光度差异反向传播到噪声估计中。这引导采样轨迹朝向严格3D一致、身份保真的输出。在多种野外图像上的实验表明，SplatShot生成的3D化身具有优越的身份保持、照片真实感和多视角一致性。

## Abstract
Reconstructing a photorealistic 3D face avatar from a single unconstrained photograph is challenging: feed-forward 3D Gaussian Splatting (3DGS) models degrade on out-of-distribution inputs, while pretrained diffusion models produce high-fidelity images but lack multi-view consistency. We observe that these paradigms are fundamentally complementary: explicit 3D representations guarantee geometric consistency, whereas 2D diffusion priors ensure photorealism. Building on this, we propose SplatShot, a training-free framework that couples these representations directly within the denoising process. Given a base 3DGS face model and a single reference image, we jointly denoise all target views using a per-step 3D feedback loop. At each timestep, we predict clean images from the noisy latents, refit the 3DGS to these multi-view predictions, and back-propagate the photometric discrepancy between the 3DGS re-renderings and 2D predictions into the noise estimate. This steers the sampling trajectory toward strictly 3D-coherent, identity-faithful outputs. Experiments on diverse in-the-wild images demonstrate that SplatShot produces 3D avatars with superior identity preservation, photorealism, and multi-view consistency.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：从一张在野外（无约束）的照片中重建出逼真的3D人脸化身是一项极具挑战的任务，要求同时具备较高的几何一致性、身份保真度和照片真实感。
- **现有方法的局限**：
  - **显式3D表示（如3DGS、NeRF）**：通过每对象优化能达到高质量，但需要密集的多视角捕获；前馈3DGS变体（如GAGAvatar、FastAvatar）降低了捕获要求，但在分布外（in-the-wild）输入上泛化能力差。
  - **2D扩散模型**：能生成高保真度单张图像，但缺乏多视角一致性，独立采样各视角会导致严重的不一致。
  - **得分蒸馏采样（SDS）**：利用2D先验引导3D生成，但容易产生过饱和、低多样性的输出，且难以平衡身份保真与几何合理性。
- **核心观察**：显式3D表示（保证几何一致性）与2D扩散先验（保证照片真实感）本质上是互补的，若能强制它们在生成过程中互相指导，则可同时获得高质量和严格的3D一致性。
- **本文目标**：提出SplatShot，一种**无需训练**的框架，将3DGS与扩散模型直接在去噪过程中耦合，实现从单张非约束照片生成高保真、多视角一致的3D人脸化身。

## 2. 方法论

### 核心思想
- 在DDIM去噪的每个时间步，执行一个3D反馈循环：
  1. 从当前噪声潜变量预测干净图像（多视角独立生成）。
  2. 将这些多视角预测用于重新拟合（refit）一个基础3DGS模型（来自NeRSemble现有模型）。
  3. 重渲染更新后的3DGS，并计算重渲染与扩散预测之间的光度损失。
  4. 将该损失的梯度反向传播到噪声估计中，调整噪声，使采样轨迹朝向3D一致、身份保真的方向。

### 关键技术细节

1. **基础3DGS模型选择**：预存300个基础3DGS模型（来自NeRSemble数据集，20万高斯），通过DINO ViT特征在发、皮肤、整体脸型三个区域加权匹配（权重0.4/0.3/0.3），选择与输入最匹配的模型作为初始化。

2. **迭代去噪流程**（算法1）：
   - 渲染基础模型的V个视图（V=16），编码到潜空间。
   - 设置起始时间步 $t_s = \lfloor s \cdot T \rfloor$（$s=0.6$, $T=1000$），从部分噪声开始。
   - 对每个视图独立预测噪声 $\epsilon^{t,v}_\theta$。
   - 使用**噪声混合**机制：$\epsilon^{t,v}_{\text{mix}} = w \cdot \epsilon^{t,v}_\theta + (1-w) \cdot \epsilon^v_{\text{gt}}$（$w=0.4$），混合预测噪声和初始添加的真实噪声，避免早期步骤预测的脏图像破坏3DGS拟合。
   - 解码混合噪声得到预测干净图像 $\hat{I}^v$。
   - 用这些图像通过L1+SSIM损失（式1）更新3DGS模型（N次：首次1000步，后续200步）。
   - 计算当前3DGS重渲染与预测图像的光度损失 $g = \mathcal{L}_{\text{photo}}(\hat{I}^v, \mathcal{R}(M, v))$。
   - 反向传播调整噪声：$\epsilon^{t,v}_\theta \leftarrow \epsilon^{t,v}_\theta - \lambda \cdot \partial g / \partial \epsilon^{t,v}_\theta$（$\lambda=50000$）。
   - 使用调整后的噪声更新下一时间步的潜变量。
   - 去噪结束时，最终3DGS模型即为输出。

3. **相关公式**：
   - DDIM噪声添加：$x_t = \alpha_t x_0 + \sigma_t \epsilon_{\text{gt}}$
   - 预测干净图像：$\hat{x}_0 = (x_t - \sigma_t \epsilon_\theta) / \alpha_t$
   - 3DGS渲染：$\hat{I} = \mathcal{R}(M, v)$
   - 光度损失：$\mathcal{L}_{\text{photo}} = \lambda_1 \|I - \hat{I}\|_1 + \lambda_2 (1 - \text{SSIM}(I, \hat{I}))$

## 3. 实验设计

### 数据集与场景
- **训练/基线数据**：来自NeRSemble数据集的300个基础3DGS模型（多视图序列）。
- **测试**：随机选取**6279张**图片来自CelebA和FFHQ数据集，覆盖各种年龄、种族、姿态、光照、配饰。
- **量化评估子集**：100个FFHQ身份，每个身份渲染16个视图。

### 基准方法对比
- 对比了三类方法：
  - **单图像→3DGS**：LAM, DreamGaussian, GAGAvatar, FastAvatar
  - **扩散方法**：Human-3Diffusion, FaceLift, Arc2Avatar
  - **3DGS编辑**：Intergsedit（重新实现为图像指导的编辑）
- **评价指标**：
  - 身份保真：CSIM（ArcFace余弦相似度，参考图像与近正面视图）
  - 多视角一致性：CV-CSIM（各视图间ArcFace相似度）
  - 几何精度：AKD（平均3D地标距离，DECA估计），AED（表情系数漂移）
  - 照片质量：FID（vs FFHQ），CLIP-IQA（无参考感知质量）

## 4. 资源与算力

- 文中明确说明：
  - 所有实验在**单张NVIDIA A100 GPU**上运行。
  - 端到端每身份耗时约**3分钟**（包括预处理、生成、评估视图渲染）。
  - 峰值内存约**24GB**。
  - 定量评估（100个身份）共需约**5 A100 GPU小时**；全6279张评估约需**314 A100 GPU小时**。
- 相比其他方法：比前馈方法慢（FastAvatar~3秒，GAGAvatar~107秒），但比优化/扩散基线快（Human-3Diffusion~5分钟，Arc2Avatar~3.5小时）。

## 5. 实验数量与充分性

- **主要定量实验**：在100个FFHH身份上比较了9种方法，报告了6个指标。SplatShot在CV-CSIM、AKD、FID、CLIP-IQA上取得最佳，CSIM排名第二（次于GAGAvatar）。
- **消融实验**：在20个身份上对5个关键设计（指导权重λ、噪声混合权重w、去噪强度s、视图数V、重拟合步数、ControlNet有无）进行消融，均展示了敏感性并确认了最佳设定。
- **基模型影响分析**：固定基模型变输入、固定输入变基模型，验证了身份转移的鲁棒性以及发型继承特性。
- **人口公平性**：按种族和年龄组计算CSIM，结果均接近（0.66~0.72），表明无明显偏见。
- **额外实验**：尝试加速采样的PeRFlow（4步），但导致多视图不一致（作为失败案例报告）。也尝试了引导正向扩散（边际改进但耗时加倍，故未采纳）。
- **充分性评价**：实验设计系统，覆盖了主要基线的全面对比，消融覆盖了核心组件，并提供了定性示例和失败案例分析。但定量仅在100个FFHQ身份上，全6279张仅用于定性展示，FID等指标可提供更全面的统计。总体上实验客观且充分。

## 6. 主要结论与发现

1. SplatShot能在无需训练的条件下，从单张非约束照片生成**同时具有高身份保真度、照片真实感和严格多视角一致性**的3DGS人脸化身。
2. 相比现有方法（前馈3DGS、扩散方法、SDS），它在多数指标上达到或超越当前最优，特别是多视角一致性（CV-CSIM 0.832，第二高0.811）和感知质量（CLIP-IQA 0.633，第二高0.621）。
3. **无需训练且即插即用**：利用现成的预训练扩散模型和基础3DGS模型，无需微调。
4. 噪声混合机制是关键，确保了早期去噪步骤的稳定性；3DGS反馈循环不断校正扩散预测的几何不一致性。
5. 基础模型的选择影响最终发型和几何细节，但身份可以通过去噪有效迁移。

## 7. 优点

- **无需训练**：不需要对任何模型组件进行额外训练，直接利用off-the-shelf的预训练模型。
- **巧妙耦合3D与2D先验**：在去噪循环内建立双向反馈，使扩散模型和3DGS互相改进，而不是单向利用（如SDS）。
- **噪声混合机制**：有效解决早期步骤预测图像质量差的问题，防止3DGS崩溃。
- **泛化能力强**：在广阔多样的野外图像上表现良好，且人群公平性较好。
- **充分消融与论证**：通过系列实验验证了各组件贡献，并解释了为什么跨注意力注入方法（如Intergsedit）在图像条件化下不适用。

## 8. 不足与局限

- **速度较慢**：每身份约3分钟，远慢于前馈方法（秒级），不适合实时应用。
- **几何依赖基础模型**：基础3DGS来自NeRSemble（仅覆盖前180度、无帽子示例），因此后视图生成未评估，且帽子等配件常失败。
- **发型继承**：最终输出中发型主要继承自基础模型，而非输入照片，因为身份编码器通常裁剪掉头发区域。
- **实验规模有限**：定量实验仅基于100个FFHQ身份，虽展示了趋势，但统计显著性未提及；跨数据集（CelebA）仅在定性示例中展示。
- **可控性不足**：缺乏对表情、视角等参数的直接控制（但论文声明专注于几何和身份重建）。
- **潜在滥用风险**：易用于生成未经授权的数字复制品或深度伪造。作者建议部署时加入同意机制，并在系统中嵌入水印。
- **可复现性**：代码已开源，但基础模型依赖NeRSemble数据集的特定模型，可能不易完全复现。
