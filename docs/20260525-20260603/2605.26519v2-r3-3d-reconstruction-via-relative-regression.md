## 摘要
最近的前馈几何基础模型通过在前向传播中恢复深度和姿态，展示了令人印象深刻的泛化能力。然而，这些模型通常受限于全局坐标框架假设。这一依赖成为长上下文和流式重建的重大瓶颈，因为它迫使网络维持一个任意的时间原点，并处理随时间无界增长的平移幅度。我们的解决方案称为$R^3$，采用相对回归。我们使用一个轻量级MLP来预测置信度加权的相对约束。这些置信度作为统一锚点：在训练时加权损失，在推理时指导姿态聚合。$R^3$支持全上下文离线重建以及因果、有界内存的流式处理。我们在离线与流式场景下的评估验证了相对机制的有效性。项目页面：https://kevinxu02.github.io/r3-site

## Abstract
Recent feed-forward geometry foundation models have demonstrated impressive generalization by recovering depth and poses in a single forward pass. However, these models are typically constrained by a global coordinate frame assumption. This dependency becomes a significant bottleneck for long-context and streaming reconstruction, as it forces the network to maintain an arbitrary temporal origin and handle translation magnitudes that grow unbounded over time. Our solution, which we call $R^3$, employs relative regression. We employ a lightweight MLP to predict confidence-weighted relative constraints. These confidences serve as a unified anchor: weighting losses during training and guiding pose aggregation during inference. $R^3$ supports both full-context offline reconstruction and causal, bounded-memory streaming. Our evaluation in both offline and streaming settings validates the effectiveness of our relative mechanism. Project page: https://kevinxu02.github.io/r3-site

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的前馈几何基础模型（如 DUSt3R、VGGT、π³、DA3）通过单次前向传播即可恢复深度和相机姿态，但它们普遍依赖**全局坐标框架**（例如固定第一帧为原点）。在面向长序列和流式重建时，这种设计迫使网络维持一个人为的时间原点，并处理随时间无限增长的平移幅度，成为严重的性能瓶颈。
- **研究动机**：作者认为，前馈三维重建应当学习**视图间的局部关系**，再通过下游步骤组装全局轨迹。相对姿态作为学习目标具有天然优势：其尺度仅取决于两帧之间的基线，不随序列总长度增长；同时 N 帧序列可产生 O(N²) 个成对约束，比 N 个绝对姿态更丰富。
- **整体含义**：本文提出 **R³（3D Reconstruction via Relative Regression）**，将重建任务重新表述为**成对相对姿态回归**，并引入**可学习的置信度**用于训练加权、推理聚合以及关键帧管理，从而同时支持离线全上下文重建和因果、有界内存的流式重建。

### 2. 论文提出的方法论

- **核心思想**：放弃直接回归全局绝对姿态，改为**预测每对帧之间的相对变换**及其对应的旋转/平移置信度，再通过置信度加权的融合算法组装全局轨迹。
- **关键技术细节**：
  - **骨干网络**：基于 DA3（Depth Anything 3）骨干，每个帧提取一个**相机 token**（\(z_i\)）。
  - **成对姿态头**：一个轻量 MLP，输入一对相机 token \((z_i, z_j)\)，输出：
    - 相对旋转（单位四元数）\(\hat{q}_{i→j}\)
    - 相对平移（j 在 i 坐标系下的平移）\(\hat{t}_{i→j}\)
    - 旋转置信度 \(c^R_{i→j}\) 和平移置信度 \(c^T_{i→j}\)
  - **训练损失**：采用**置信度加权的 L1 残差**，并添加对数正则项（\(\alpha=0.2\)）防止置信度坍缩。
    \[
    \mathcal{L}_{rot}=c^R_{i→j}\ell_{L1}^{rot} - \alpha \log c^R_{i→j},\quad \mathcal{L}_{trans}=c^T_{i→j}\ell_{L1}^{trans} - \alpha \log c^T_{i→j}
    \]
    深度损失类似，用中值归一化的深度和像素级置信度。
  - **推理时的轨迹聚合**：
    - 第一帧固定为原点。
    - 对新帧 \(j\)，每个参考帧 \(i\) 提出候选姿态（组合已估计的 \(T_i\) 与预测的相对姿态）。
    - 最终姿态为候选姿态的**置信度加权融合**（旋转和翻译分别软最大化加权）。
    - **关键帧库**：管理数量 \(M_{\max}=100\) 的活跃上下文，使用**特征新颖性**（余弦相似度）决定是否将新帧加入，最低效用（“差异性×置信度”）的帧被驱逐，第一帧不可驱逐。
  - **流式与全上下文**：同一检查点通过**测试时切换因果掩码**即可切换模式。全上下文模式下额外执行一次轻量置信度加权位姿图优化（仅优化位姿，不涉及深度或重投影）。
- **算法流程**（文字说明）：
  1. 输入视频流帧。
  2. 骨干网络（因果/全注意力）输出每帧相机 token 和深度图。
  3. 对每对帧（流式中仅对新帧与关键帧库中的帧），MLP 预测相对姿态及置信度。
  4. 通过置信度加权聚合得到新帧的全局位姿。
  5. 根据特征新颖性和置信度更新关键帧库。
  6. （可选）全上下文模式：去除因果掩码，利用所有帧间边缘进行位姿图优化。

### 6. 论文的主要结论与发现

- **相对姿态回归**是一种更强的基础几何先验，尤其适合长序列和流式场景：它解除了对固定全局原点的依赖，且平移量有界。
- **解耦的旋转/平移置信度**可灵活用于训练损失、推理聚合、关键帧管理、离群检测等多个模块，形成统一的置信驱动管道。
- R³ 在**流式重建**中达到 SOTA 或接近 SOTA 的精度，同时内存消耗有界、计算开销低（372M 参数，>20 FPS）。
- 同一模型可**无缝切换**为离线全上下文模式，无需重新训练。
- 置信度机制天然具备**异常帧抑制**能力，在干扰帧注入实验中展现出极高的鲁棒性。

### 7. 优点

- **创新性**：首次将前馈重建问题系统性地转化为置信加权相对回归，彻底摆脱全局坐标框架。
- **轻量高效**：仅 372M 参数，6 块 48GB GPU 即可训练，推理速度快，适合实际部署。
- **统一置信信号**：训练、推理、关键帧管理、离群检测均依赖同一组置信度，设计简洁且有效。
- **灵活的双模式**：单检查点支持因果流式和全上下文推理，适用性广。
- **强鲁棒性**：单点、低复杂度离群检测机制性能优于需要多遍离线检测的 RobustVGGT。

### 8. 不足与局限

- **人工阈值依赖**：关键帧库的入库阈值 \(\tau\)、最大间隙 \(\Delta_{\max}\)、容量 \(M_{\max}\)、离群门限等仍需手动设定，在不同场景下可能需重新调整。
- **长期漂移**：即使有置信度机制，超长序列（如 >1000 帧）仍可能出现漂移，论文承认需要周期性重置（segment reset）。
- **因果流式的固有缺陷**：无法向后看到未来帧，导致远处视角或严重遮挡时残差残留，且未探索全局双向重注意力或联合深度‑位姿优化。
- **未探索更大规模**：受限于计算资源，未在更大骨干（如 1B+ 参数）或更丰富数据集上验证，性能上限未知。
- **点云重建指标仍落后部分离线方法**：例如在 NRGBD 上的 NC 指标低于 STream3R 等。
