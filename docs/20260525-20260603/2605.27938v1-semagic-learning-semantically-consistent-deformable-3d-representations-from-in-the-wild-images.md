## 摘要
从单视图野外图像中学习可变形3D物体模型已实现了无需监督的令人印象深刻的3D形状重建。然而，这些模型是否捕获了下游任务所需的语义结构仍不清楚。我们发现，现有的可变形重建方法尽管生成视觉上合理的几何形状，但在实例间产生不稳定的对应关系，并且在语义对应基准上表现不佳。我们提出了SEMAGIC，一个从单视图野外图像中学习语义一致的可变形3D表示的框架。SEMAGIC不将重建视为最终目标，而是将可变形建模作为发现类别级对应关系的机制。每个类别由一个规范模板网格和一个学习到的变形场表示，其功能类似于自编码器，从图像特征重建实例几何形状，使得顶点在实例间保持一致的语义含义。训练期间通过（i）特征级一致性损失，对齐规范网格和变形网格之间的语义特征，以及（ii）顶点索引条件变形，保持跨实例的语义对应关系，来强制语义一致性。通过明确地将几何变形与语义对齐耦合，SEMAGIC生成的表示在类别内变化中保持稳定的部件对应关系。实验表明，SEMAGIC在SPair-71k上将可变形模型的语义对应关系提高了+14.7 PCK@0.1，确立了可变形模型作为有效语义3D表示的地位。

## Abstract
Learning deformable 3D object models from single-view in-the-wild images has enabled impressive 3D shape reconstruction without supervision. However, it remains unclear whether these models capture the semantic structure required for downstream tasks. We find that existing deformable reconstruction approaches, despite producing visually plausible geometry, yield unstable correspondences across instances and perform poorly on semantic correspondence benchmarks. We introduce SEMAGIC, a framework for learning semantically consistent deformable 3D representations from single-view in-the-wild images. Rather than treating reconstruction as the end goal, SEMAGIC uses deformable modeling as a mechanism to discover category-level correspondences. Each category is represented by a canonical template mesh and a learned deformation field, functioning similarly to an autoencoder that reconstructs instance geometry from image features, enabling vertices to maintain consistent semantic meaning across instances. Semantic consistency is enforced during training through (i) a feature-level consistency loss aligning semantic features between canonical and deformed meshes, and (ii) vertex-index-conditioned deformation that preserves semantic correspondence across instances. By explicitly coupling geometric deformation with semantic alignment, SEMAGIC produces representations that maintain stable part correspondences across intra-category variation. Experiments demonstrate that SEMAGIC improves semantic correspondence of deformable models by +14.7 PCK@0.1 on SPair-71k, establishing deformable models as effective semantic 3D representations.

## 2. 方法论

- **核心思想**：将可变形建模视为发现类别级语义对应的机制，而非单纯的重建工具。每个类别由一个规范模板网格（canonical template mesh）和一个图像条件变形场表示，类似于自编码器结构。
- **关键技术细节**：
  - **3D形状表示**：使用混合符号距离场（hybrid SDF）结合 DMTet（可微分 Marching Tetrahedra）将隐式场实时转换为显式网格。
  - **变形网络**：MLP $F_{def}: \mathbb{R}^3 \times \mathbb{R}^{df} \times \mathbb{R}^8 \rightarrow \mathbb{R}^3$ 预测每个顶点的偏移，输入包括规范顶点坐标 $v_i$、全局图像特征 $f_I$、可学习的顶点索引嵌入 $e_i = E_\psi(i)$。
    - 公式：$\Delta v_i = F_{def}(v_i, f_I, e_i)$，变形后顶点 $v'_i = v_i + \Delta v_i$。
  - **语义一致性机制**：
    - 顶点索引条件变形：每个规范顶点具有固定的索引嵌入，共享于所有实例，鼓励每个顶点学习一致的语义角色（如椅子腿、靠背）。
    - 特征级一致性损失：使用坐标特征 MLP $F_{feat}$ 将规范网格和变形网格的对应顶点映射到高维语义嵌入，通过 L2 损失 $\mathcal{L}_{const} = \frac{1}{N}\sum_i \|f_i^{prior} - f_i^{ins}\|^2$ 强制它们相似。
  - **几何感知语义对应检索**：
    - 利用规范网格与变形网格的密集 3D 对应关系 $C_{3D}: v_i \leftrightarrow v'_i$。
    - 将变形顶点投影到图像平面并过滤自遮挡顶点，对于源关键点找到最近可见顶点，再在目标图像中进行局部特征搜索（24×24 窗口）以获得精细匹配。
  - **视觉基础模型监督**：为处理刚性物体类别（缺乏关节、对称性强、视角变化大）引入：
    - 深度监督：使用 Depth Anything V2 预测的深度图，采用平移缩放不变的 L2 损失 $\mathcal{L}_{depth} = \frac{1}{|mask|}|\alpha D_{pred}+\beta - D_{gt}|^2$。
    - 姿态监督：使用 Orient Anything V2 预测姿态，采用余弦回归损失 $\mathcal{L}_{pose} = 1 - \cos(\theta_{pred} - \theta_{gt})$ 解耦姿态与变形。

## 6. 主要结论与发现

- SEMAGIC 在 SPair-71k 上平均 PCK@0.1 达到 52.1，比 MagicPony（37.4）提升 14.7 个点；即使 MagicPony 加上深度和姿态监督（43.9），SEMAGIC 仍领先 8.2 个点。

- 姿态监督对刚性物体关键：移除 pose 后 PCK@0.1 下降至 44.7，表明解耦姿态与变形是前置条件。
- 定性结论：SEMAGIC 的规范空间顶点颜色分布更一致，同一语义部位在不同实例上具有相同颜色；MagicPony 则出现明显色块不一致（如机翼、船体、车轮等）。
- 3D 重建质量也有提升：归一化 Chamfer 距离从 8.2 降至 6.2（相对改善 24%），说明语义约束也有助于几何保真度。
