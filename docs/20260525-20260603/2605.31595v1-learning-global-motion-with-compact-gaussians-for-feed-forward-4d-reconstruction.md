## 摘要
从单目视频进行动态场景重建仍然是计算机视觉中的一个基本挑战。现有的前馈方法为每帧逐像素预测3D高斯，但存在重复高斯和视角依赖偏差的问题，阻碍了场景运动的有效学习。我们提出C4G，一个基于紧凑的时间戳条件可学习高斯查询令牌集的前馈4D重建框架。每个令牌聚合整个时间上下文中对应的特征，并解码出一个3D高斯，其位置由目标时间戳调制，实现了无需逐场景优化的全局一致运动建模。为捕捉细粒度细节，我们进一步引入基于视频扩散模型的渲染增强模块。由于我们的框架有效地将特征聚合到高斯中，我们将这一能力扩展至特征提升，生成支持点追踪和动态场景理解的4D特征场。C4G在显著减少高斯数量且无需相机位姿的情况下，实现了强的新视角合成性能，同时展现出更强的运动建模能力以及对大时间间隔的鲁棒性。

## Abstract
Dynamic scene reconstruction from monocular video remains a fundamental challenge in computer vision. Existing feed-forward methods predict 3D Gaussians pixel-wise for each frame, suffering from duplicated Gaussians and view-dependent biases that hinder effective learning of scene motion. We present C4G, a feed-forward 4D reconstruction framework built upon a compact set of timestamp-conditioned learnable Gaussian query tokens. Each token aggregates corresponding features across the full temporal context and decodes a 3D Gaussian whose position is modulated by the target timestamp, enabling globally coherent motion modeling without per-scene optimization. To capture fine-grained details, we further introduce a video diffusion model-based rendering enhancement module. Since our framework effectively aggregates features into Gaussians, we extend this capability to feature lifting, producing a 4D feature field that supports point tracking and dynamic scene understanding. C4G achieves strong novel-view synthesis performance using significantly fewer Gaussians and without requiring camera poses, while exhibiting stronger motion modeling and robustness to large temporal gaps.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：从单目视频进行动态场景4D重建，现有前馈方法（如4DGT、MoVieS、NeoVerse）采用**逐像素预测3D高斯**，但存在两大缺陷：
  - **重复高斯（Ghosting）**：在插值时间戳渲染时，从邻近输入视图产生的两组高斯位置不一致，导致虚影。
  - **视角依赖偏差**：模型过度依赖时间上最近的输入视图，无法利用远端帧信息填补遮挡区域，本质源于像素级预测带来的过度表示能力，使模型倾向于过拟合输入视图而非学习全局运动。
- **动机**：打破逐像素范式，通过**紧凑的可学习查询令牌**强制模型从全局时间上下文中聚合信息，学习连贯的场景运动，同时保持高效和高渲染质量。

## 2. 方法论

### 2.1 核心思想
- 用**固定数量（N=2048）的可学习查询令牌**替代逐像素高斯。
- 每个令牌通过Transformer解码器与多帧视觉特征交互，并**条件于目标时间戳**，解码出一个3D高斯。
- 通过变化目标时间戳，一组令牌即可表示任意时刻的动态场景，实现紧凑、全局运动一致的4D表示。

### 2.2 关键技术细节

#### 2.2.1 4D高斯重建模块（4D Gaussian Reconstruction Module）
- **视觉特征提取器**：基于预训练的VGGT编码器，为每帧提取几何感知特征。
- **时间嵌入**：
  - 对上下文帧的特征添加其归一化时间戳（相对目标时间戳的偏移）的正弦位置编码。
  - 对可学习查询令牌添加目标时间戳编码，使解码出的高斯位置受目标时间调制。
- **查询解码器**：将时间嵌入后的特征和查询拼接，经过L=2层Transformer（全自注意力），每个查询被映射为3D高斯（均值、不透明度、协方差、颜色，颜色用0阶球谐即RGB）。
- **紧凑性**：仅2048个高斯（对比像素方法通常~802K），迫使模型用相同令牌表示静动态区域，学习运动而非记忆帧。

#### 2.2.2 VDM渲染增强模块（VDM Refinement Module）
- 采用Wan2.1-VACE-1.3B扩散模型，以C4G渲染的视频为“反应式”输入，原始上下文帧为“非反应式”参考，通过ControlNet结构进行条件去噪，增强高频细节。
- 训练数据：Kubric合成数据集，训练50K步。

#### 2.2.3 损失函数
- **色彩损失**：MSE + LPIPS。
- **深度损失**：基于MoGe-2预估伪深度，使用尺度-平移不变L1损失。
- **法线损失**：从渲染深度计算法线与MoGe-2法线的L1损失。
- **追踪损失**：基于CowTracker的点轨迹Huber损失，约束高斯投影位置的时序一致性。
- **动态掩码权重**：对动态区域施加更高损失权重（λ_dynamic=3 效果最佳）。

### 2.3 任意特征4D提升（Any-feature 4D Lifting）
- 利用C4G解码器中已学到的注意力模式（第一层广泛关注空间对应区域，第二层聚焦邻近帧），重用一个结构相同的特征解码器，仅训练值投影层，将任意视觉基础模型的特征（如DINOv3、VGGT、LSeg）提升到每个高斯上，形成4D特征场。
- 训练仅需5K步，因为注意力模式已就绪。

## 6. 主要结论与发现

1. **紧凑查询大幅优于像素级预测**：仅用2K高斯（0.007×或更少），在所有动态数据集上达到SOTA或同等性能，特别在DyCheck和TUM上领先明显。
2. **无需相机位姿**：C4G完全无位姿依赖，且性能优于需位姿的基线。
3. **全局运动理解**：通过分析注意力图，第一层捕获空间对应，第二层聚焦时间邻近帧，体现了隐式运动建模。
4. **对时间间隔鲁棒**：随着∆t增加，其他方法性能大幅下降，C4G仅轻微下降。
5. **4D特征场有效**：提升后的特征在点追踪和场景理解中超越原始2D特征，表明模型具备时空一致的表示能力。
6. **VDM增强有效**：在几何一致性指标（背景一致性、时间闪烁、运动平滑度）上超过先前扩散方法。

## 7. 优点

- **架构创新**：用可学习查询替代逐像素预测，从根本上解决了重复高斯和视角偏差问题，是前馈4D重建的新范式。
- **高效紧凑**：2048个高斯即可达到高质量重建，推理速度快且内存友好。
- **无需位姿**：摆脱了对相机位姿的依赖，适用于大量无位姿的真实视频。
- **多任务扩展**：天然支持特征提升，可直接用于点追踪、语义分割等下游任务。
- **可解释性**：注意力可视化展示了模型如何隐式学习空间与时间对应，利于分析和调试。
- **实验扎实**：涵盖多个难度差异的数据集、多种评价协议、充分消融。

## 8. 不足与局限

- **视角外推能力弱**：只能重建在输入帧中至少出现一次的区域；极端新视角下会产生空洞，需依赖VDM填充，但VDM会引入幻觉。
- **时间外推无效**：模型仅训练于插值时间戳，无法准确预测未来帧高斯，限制了视频外推应用。
- **VDM训练数据有限**：仅使用Kubric合成数据，可能导致在真实场景细粒度细节恢复上不够好（表7中图像质量分数相对较低）。
- **未公开训练总时长**：缺少关于4个GPU需要多少小时或天数的信息，增加复现难度。
- **尺度归一化依赖**：无需位姿但需测试时对齐（因为无绝对尺度），对齐过程可能引入误差。
- **动态区域监督仍显不足**：虽然使用了动态掩码加权，但单目视频中的运动模糊和遮挡仍可能限制几何精度。
- **对比方法选择**：未对比近期如StreamSplat、4D-LRM等，可能遗漏更强基线；对于需要位姿的方法，C4G在无位姿下比较已占优，但更公平的对比应与其他无位姿方法（如NeoVerse）充分对比。
