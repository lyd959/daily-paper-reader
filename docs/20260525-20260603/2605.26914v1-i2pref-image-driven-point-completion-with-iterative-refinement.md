## 摘要
我们提出了一种图像条件点云补全方法，该方法将图像视为主要几何源而非辅助引导。为此，我们引入了一个图像到点（I2P）模块，可以直接从单个RGB图像重建完整的点云，无需3D输入。此外，我们还引入了一个基于Transformer的点到点（P2P）精细化模块，利用点令牌与图像特征之间的自注意力和交叉注意力来迭代地精细化粗糙的I2P输出。I2P模块使图像编码器能够学习丰富的几何表示，而P2P模块则逐步恢复细节。与依赖辅助损失或融合模块的现有多模态方法不同，我们明确的I2P任务提供了仅基于图像的强几何感知先验。在ShapeNet-ViPC上的大量实验表明，与先前方法相比，我们的方法实现了最先进的补全性能，相对倒角距离提升了12.3%。代码可在以下网址获取：https://github.com/AzharSindhi/I2PRef.git

## Abstract
We present an image-conditioned point cloud completion approach that treats images as the primary geometric source rather than a secondary guide. To this end, we introduce an Image-to-Point (I2P) module that can reconstruct complete point clouds directly from a single RGB image, with no need for 3D inputs. Additionally, we introduce a transformer-based Point-to-Point (P2P) refinement module that uses self- and cross-attention between point tokens and image features to iteratively refine the coarse I2P output. The I2P module enables the image encoder to learn rich geometric representations, while the P2P module progressively recovers fine-grained details. Unlike existing multimodal methods that rely on auxiliary losses or fusion modules, our explicit I2P task provides a strong, geometry-aware prior based on images alone. Extensive experiments on ShapeNet-ViPC demonstrate state-of-the-art completion performance with a 12.3% relative Chamfer Distance improvement over prior methods. Code is available at: https://github.com/AzharSindhi/I2PRef.git

## 1. 核心问题与整体含义（研究动机与背景）

- **研究背景**：点云补全是3D视觉中的重要任务，旨在从不完整的扫描中恢复完整几何形状。真实场景中的点云常因遮挡、传感器噪声、视角限制而不完整。传统方法仅依赖3D点云输入，而多模态方法（如ViPC、XMFNet、EGIInet）尝试引入单张RGB图像作为辅助信息，但通常将图像视为次要引导，通过辅助对齐损失或融合模块来利用图像特征，未能充分发挥图像中蕴含的丰富几何信息。
- **核心问题**：如何将单张RGB图像作为**主要几何源**，直接生成完整的点云，并通过迭代细化提升细节质量。
- **论文含义**：提出一种新范式——显式地从图像到点云（I2P）生成粗点云，再通过迭代的Transformer细化（P2P）恢复精细结构。这种方法避免了复杂的跨模态融合设计，仅依赖图像本身提供强几何先验，在ShapeNet-ViPC基准上实现了SOTA性能（CD相对提升12.3%）。

## 2. 方法论：核心思想、关键技术、公式与流程

### 核心思想
将点云补全分解为两个阶段：
1. **I2P模块（Image-to-Points）**：直接从单张RGB图像生成一个完整的粗点云，作为几何先验。
2. **P2P模块（Point-to-Point）**：基于Transformer的迭代细化网络，利用自注意力和交叉注意力（与图像特征）逐步修正点位置，提升细节。

### 关键技术细节
- **I2P模块**：
  - 采用U-Net风格的层次编码器（类似扩散模型骨架），包含7×7卷积、多个下采样阶段（每个阶段含两个ResNet块+RMSNorm+SiLU激活+注意力层+空间下采样），瓶颈处加全自注意力，生成特征图F ∈ R^(H'×W'×C)。
  - 点生成器：采用类似XMFNet的架构，将图像特征聚合为全局隐变量，解码生成N_g=1024个新点，然后与从输入部分点云均匀下采样的1024个点拼接，得到初始粗点云P0（共2048点）。
- **P2P模块**：
  - 将粗点云坐标线性投影为嵌入X0 ∈ R^(N×D)。
  - 经过L=4次迭代，每次迭代使用一个Transformer块，包含：
    - 自注意力：点嵌入之间。
    - 交叉注意力：点嵌入作为查询，图像特征FI作为键/值。
    - 前馈网络+残差连接。
  - 从更新后的嵌入通过轻量MLP预测坐标偏移ΔP^(l)，更新点坐标：P^(l+1)=P^(l)+ΔP^(l)。最终Pc=P^(L)。
- **损失函数**：联合监督粗输出和最终输出，Chamfer Distance：L_cd = CD(Pc, Pgt) + α·CD(P0, Pgt)。其中α从0.7线性退火至0.1。

### 算法流程（文字说明）
1. 输入单张RGB图像I，通过I2P模块的U-Net编码器得到图像特征FI。
2. 点生成器从FI生成1024个新点，与部分点云下采样后的1024个点拼接，得到粗点云P0。
3. 将P0线性投影为嵌入，进入P2P模块；对于l=1..L迭代：
   - 执行自注意力、交叉注意力、FFN，更新嵌入；
   - 预测偏移并累加，得到P^(l+1)。
4. 输出最终点云Pc。
