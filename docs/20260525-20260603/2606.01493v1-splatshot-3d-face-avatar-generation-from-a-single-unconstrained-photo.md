## 摘要
从单张无约束照片重建逼真的3D人脸头像具有挑战性：前馈式3D高斯溅射（3DGS）模型在分布外输入上退化，而预训练扩散模型生成高保真图像但缺乏多视角一致性。我们观察到这些范式本质上是互补的：显式3D表示保证几何一致性，而2D扩散先验确保照片真实感。基于此，我们提出Splatshot，一种无训练框架，直接在去噪过程中耦合这些表示。给定一个基础3DGS人脸模型和一张参考图像，我们使用每步3D反馈循环联合去噪所有目标视图。在每个时间步，我们从噪声潜变量预测干净图像，将这些多视角预测重新拟合到3DGS，并将3DGS重新渲染与2D预测之间的光度差异反向传播到噪声估计中。这将采样轨迹引导至严格3D一致、身份忠实的结果。在多种野外图像上的实验表明，Splatshot生成的3D头像具有优越的身份保持、照片真实感和多视角一致性。

## Abstract
Reconstructing a photorealistic 3D face avatar from a single unconstrained photograph is challenging: feed-forward 3D Gaussian Splatting (3DGS) models degrade on out-of-distribution inputs, while pretrained diffusion models produce high-fidelity images but lack multi-view consistency. We observe that these paradigms are fundamentally complementary: explicit 3D representations guarantee geometric consistency, whereas 2D diffusion priors ensure photorealism. Building on this, we propose SplatShot, a training-free framework that couples these representations directly within the denoising process. Given a base 3DGS face model and a single reference image, we jointly denoise all target views using a per-step 3D feedback loop. At each timestep, we predict clean images from the noisy latents, refit the 3DGS to these multi-view predictions, and back-propagate the photometric discrepancy between the 3DGS re-renderings and 2D predictions into the noise estimate. This steers the sampling trajectory toward strictly 3D-coherent, identity-faithful outputs. Experiments on diverse in-the-wild images demonstrate that SplatShot produces 3D avatars with superior identity preservation, photorealism, and multi-view consistency.

  - 预训练扩散模型能够生成高质量单张图像，但缺乏多视图一致性；

- **本文贡献**：提出 **SplatShot**，一种**无训练**框架，在扩散去噪过程中通过**每步3D反馈循环**将显式3DGS与2D扩散先验直接耦合，从而同时获得高保真身份保持、多视图一致性和实际感。

## 2. 论文提出的方法论

### 2.1 核心思想
- 给定一个预训练的**基础3DGS模型**（来自多视角数据集NeRSemble）和一张输入照片，对多视图进行联合去噪。
- 在去噪的每个时间步，交替进行：
  1. **用扩散模型独立生成每个视图的干净图像**；
  2. **用这些预测图像重新拟合（refit）3DGS模型**；
  3. **计算3DGS重渲染与扩散预测之间的光度损失，并反向传播梯度调整噪声估计**。
- 通过这种双向信息流，扩散模型被引导至3D一致的结果，同时3DGS收敛到目标身份。

### 2.2 关键技术细节
- **噪声混合机制**：早期去噪步骤中，预测的干净图像噪声大、质量差。将扩散预测噪声与初始加入的真实噪声按权重 \(w\) 混合，得到稳定的混合噪声用于预测，既保持梯度可导又防止3DGS崩溃。
- **去噪过程**：采用DDIM确定性采样，从部分初始化噪声开始（强度 \(s=0.6\)），保留基础模型的结构线索。
- **条件控制**：使用IP-Adapter进行身份条件注入，ControlNet进行姿态控制。
- **算法流程**：
  1. 从基础模型渲染多视图（\(V=16\)）；
  2. 编码到潜在空间并加噪至起始时间步；
  3. 对每个时间步 \(t\)：
     - 每个视图独立预测噪声 → 解码得到预测图像；
     - 更新3DGS模型（梯度下降最小化光度损失）；
     - 计算重渲染与预测的光度损失 → 反向传播调整噪声；
     - 使用调整后的噪声计算下一时间步的潜在变量；
  4. 去噪结束后输出最终3DGS模型。

### 2.3 公式说明（文字）
- 混合噪声：\(\epsilon_{\text{mix}} = w \cdot \epsilon_\theta + (1-w) \cdot \epsilon_{\text{gt}}\)
- 噪声调整：\(\epsilon_\theta \leftarrow \epsilon_\theta - \lambda \cdot \partial g / \partial \epsilon_\theta\)，其中 \(g\) 是光度损失，\(\lambda\) 是引导权重。

## 6. 论文的主要结论与发现

- SplatShot能够从单张野外照片生成**高身份保真、多视图一致、照片真实感强**的3DGS头像。
- 其**无训练框架**有效结合了显式3D表示和隐式2D先验，无需任何微调或额外训练。
- 噪声混合机制是稳定早期引导的关键；3D反馈循环确保了严格的几何一致性。

## 7. 优点

- **创新性**：首次在无训练框架内，通过每步3D反馈循环将扩散模型与3DGS直接耦合，思路新颖且高效。

- **实用性**：无需多视图输入或个性化优化，输入单张照片即可生成可渲染的3D头像；代码开源。

## 8. 不足与局限

- **速度慢**：每个身份约3分钟，远慢于前馈方法（秒级），不适合实时应用。
- **几何依赖基础模型**：发型、头部形状主要继承自所选基础模型，若基础模型发型不匹配则出现偏差。
- **视角受限**：仅评估前180度（NeRSemble数据覆盖范围），未测试背视生成能力。
- **特定物体处理困难**：帽子等大型遮挡物难以正确重建（原因：基础数据集无帽子样本，身份编码器将帽子混入身份特征）。
- **可扩展性**：需要预存一组基础3DGS模型（目前300个），若需覆盖更多几何风格需扩充。
- **应用风险**：可用于生成深度伪造或身份滥用，论文建议部署时加入同意机制和水印，但未提供具体实现协议。
